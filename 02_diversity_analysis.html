Aim of this notebook¶

This notebook investigates how fire affects the structure and diversity of microbial communities in tropical peatlands, focusing on both Archaea and Bacteria across different soil depths. The goal is to use statistical modeling and A/B testing (burnt vs. intact peat) to identify where and how microbial balance, diversity, and community composition shift after fire. The workflow demonstrates how to move from raw community data to interpretable results using both univariate and multivariate methods, with an emphasis on effect sizes and practical significance. This approach is directly relevant to data scientists interested in experimental design, compositional data, and robust group comparisons in high-dimensional biological datasets.

Compute Archaea:Bacteria Ratio¶

Compute the Archaea:Bacteria ratio by aggregating counts to the Kingdom level, converting to relative abundances, and dividing archaeal by bacterial proportions per sample. This ratio serves as a simple but powerful indicator of broad community balance, helping to identify whether fire shifts the dominance between these two major microbial groups. In industry terms, this is akin to tracking a key performance indicator (KPI) that summarizes a complex system in a single, interpretable metric.

In [5]:
## compute archea:bacteria ratio
archaea_to_bacteria_ratio <- btp_fire_ps %>%
  tax_glom("Kingdom") %>%
  transform_sample_counts(., function(x) x / sum(x)) %>%
  psmelt() %>%
  select(Sample, Abundance, psf_type, depth, Kingdom) %>%
  pivot_wider(
    id_cols = c("Sample", "psf_type", "depth"),
    names_from = "Kingdom",
    values_from = "Abundance"
  ) %>%
  group_by(Sample) %>%
  mutate(ratio = Archaea / Bacteria) %>%
  ungroup

Model Ratio with Two-Way ANOVA¶

Fit a two-way ANOVA to the Archaea:Bacteria ratio, modeling burn status (A/B: burnt vs. intact), depth, and their interaction. This factorial design allows us to test not only whether fire has an effect, but also whether that effect depends on depth. The approach is directly analogous to A/B testing with stratification, and the interaction term helps reveal if the treatment effect is consistent or context-dependent.

Compute Archaeal Shannon Diversity¶

Compute archaeal Shannon diversity by subsetting Archaea, estimating per-sample diversity, and joining sample metadata. Shannon diversity captures both richness (number of types) and evenness (balance among types) in a single value. This metric is widely used in ecology and is conceptually similar to entropy or diversity indices in information theory and business analytics.

In [9]:
## compute shannon diversity for archaea
shannon_diversity_archaea <- btp_fire_ps %>%
  subset_taxa(., Kingdom == "Archaea") %>%
  estimate_richness(., measures = "Shannon") %>%
  rownames_to_column("sample_name") %>%
  inner_join(
    .,
    sample_data(btp_fire_ps) %>%
      data.frame() %>%
      rownames_to_column("sample_name")
  ) %>%
  select(sample_name, Shannon, psf_type, depth)

Joining with `by = join_by(sample_name)`

Helper: Covariance Ellipses for Ordination¶

Define a helper function to generate covariance-based ellipses for ordination plots. These ellipses visually summarize the spread and separation of groups in multivariate space, helping to interpret complex community composition data in a more intuitive way.

In [16]:
## function to compute ordination ellipses
veganCovEllipse<-function (cov, center = c(0, 0), scale = 1, npoints = 100) 
{
  theta <- (0:npoints) * 2 * pi/npoints
  Circle <- cbind(cos(theta), sin(theta))
  t(center + scale * t(Circle %*% chol(cov)))
}

Constrained Ordination (CAP) for Archaea¶

Build a constrained analysis of principal coordinates (CAP) for archaeal communities using Bray distances and a square-root relative-abundance transform, with burn status as the constraining variable. CAP is a multivariate technique that projects high-dimensional data into a lower-dimensional space, making it easier to visualize and test for group differences. This is analogous to using PCA or CCA in customer segmentation, but tailored for ecological data.

In [17]:
## archaea constrained analysis of principle coordinates
archaea_cap <- ordinate(
  btp_fire_ps %>%
    subset_taxa(., Kingdom == "Archaea") %>%
    transform_sample_counts(., function(x) sqrt(x / sum(x))),
  "CAP", "bray", ~psf_type
)

Test the archaeal CAP model with permutation-based ANOVA to quantify the effect of burn status on community composition. Permutation tests are robust to non-normality and high dimensionality, making them well-suited for microbiome data. Reporting pseudo-F, variance explained, and permutation p-values provides a comprehensive view of the strength and reliability of observed group differences.

Generate Archaeal Covariance Ellipses¶

Generate archaeal covariance ellipses around each group to show within-group variability and emphasize separation between burnt and intact across depths. Larger ellipses indicate more spread within a group, while less overlap between ellipses suggests clearer differences between groups.

In [21]:
## extract data to plot
archaea_cap_ellipse <- data.frame()
for (g in levels(archaea_cap_df$index)) {
  archaea_cap_ellipse <- rbind(archaea_cap_ellipse, cbind(as.data.frame(with(
    archaea_cap_df[archaea_cap_df$index == g, ],
    veganCovEllipse(
      cov.wt(cbind(CAP1, MDS1),
        wt = rep(1 / length(CAP1), length(CAP1)))$cov,
      center = c(mean(CAP1), mean(MDS1))
    )
  )),
    index = g
  ))
}
